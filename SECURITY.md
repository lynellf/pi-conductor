# Security Policy

## Reporting a vulnerability

Report vulnerabilities privately via GitHub Security Advisories:

https://github.com/earendil-works/pi-conductor/security/advisories/new

If that link is unreachable, Security Advisories may not be enabled on
this repository yet — please open a regular issue requesting they be
enabled and we'll respond within 5 business days.

## Response window

We aim to acknowledge reports within 5 business days and provide an
initial assessment.

## Supported versions

Only the latest tagged release published to npm is supported. Pre-release
or development versions (installed from source or a git ref) are not
covered.

## Disclosure policy

We'll coordinate a fix and disclosure timeline; please do not disclose
publicly until a patched release is available.
